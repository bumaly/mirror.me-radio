# Version History

## v1.0 — 2026-04-09

### Key Features
- Voice cloning via ElevenLabs Instant Voice Cloning (IVC)
- Audio recording and transcription via OpenAI Whisper
- Text rewriting through the "personal" psychological lens via GPT-4o
- Speech synthesis playback in the participant's cloned voice
- Six affirmation prompts covering relationships, body image, creative work, healing, love, and belonging
- Voice profile recording with guided on-screen script
- Pre-send review flow (duration display + re-record option)
- Prompt revisit with stored text and audio replay
- End-of-experience screen with artist statement
- Session transcript export to JSON
